# Practical No. 7

## Deploying a Model in Browser (TensorFlow.js)

---

## 📌 Objective

To train, save, reload, and deploy a machine learning model directly in the browser.

---

## 🧠 Features

* Train model in browser
* Save model to LocalStorage
* Reload model and verify predictions
* Export & import model files
* Compare prediction accuracy

---

## 🛠️ Technologies Used

* HTML5, CSS3, JavaScript
* TensorFlow.js (CDN)

---


---

## ⚙️ How to Run

1. Open `index.html` in browser
2. Click "Train Model"
3. Save and reload model
4. Test predictions

### 🔹 Assi 3

## 📌 Conclusion

TensorFlow.js allows full ML lifecycle in browser including training, saving, and deployment without backend support.
