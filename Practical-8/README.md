# Practical No. 8

## Transfer Learning with MobileNet (TensorFlow.js)

---

## 📌 Objective

To implement transfer learning using MobileNet for custom image classification tasks.

---

## 🧠 Features

* MobileNet feature extraction
* Custom dataset creation (images/webcam)
* Multi-class classification
* Real-time prediction
* Incremental learning support
* Confusion matrix & accuracy evaluation

---

## 🛠️ Technologies Used

* HTML5, CSS3, JavaScript
* TensorFlow.js (MobileNet via CDN)

---

## 📁 Project Structure

```
Practical-8/
│
├── index.html
├── style.css
├── model.js
│
├── Assignment/
│   ├── assi1/
│   ├── assi2/
│   ├── assi3/
```

---

## ⚙️ How to Run

1. Open `index.html` in browser
2. Add image dataset (upload/webcam)
3. Train model
4. Test predictions

---

## 📊 Assignment Tasks

### 🔹 Assi 1

* Train MobileNet on 3 classes
* Show accuracy

### 🔹 Assi 2

* Evaluate model
* Display confusion matrix

### 🔹 Assi 3

* Add new classes
* Retrain and compare performance

---

## 📌 Conclusion

Transfer learning using MobileNet enables fast and efficient custom image classification directly in the browser.
