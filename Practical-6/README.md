# Practical No. 6

## Pose Detection using PoseNet (TensorFlow.js)

---

## 📌 Objective

To detect human body poses in real-time using PoseNet and analyze movements such as exercise repetitions.

---

## 🧠 Features

* Real-time pose detection using webcam
* Keypoints and skeleton visualization
* Exercise repetition counter (Squats)
* Single-pose vs Multi-pose comparison
* Confidence score & FPS display

---

## 🛠️ Technologies Used

* HTML5, CSS3, JavaScript
* TensorFlow.js (PoseNet via CDN)
* Canvas API

---

## ⚙️ How to Run

1. Open `index.html` in browser
2. Allow webcam access
3. Start pose detection

---

## 📊 Assignment Tasks

### 🔹 Assi 1

* Detect keypoints
* Draw skeleton

### 🔹 Assi 2

* Squat counter using knee angles
* Display reps & posture

### 🔹 Assi 3

* Compare single vs multi pose
* Show accuracy & person count

---

## 📌 Conclusion

PoseNet enables real-time pose detection efficiently in browser. It can be used for fitness tracking, gesture recognition, and motion analysis applications.
